# 🚀 快速开始

```shell
git clone <url_to_project>
cd <path_to_project>
npm install
npm run tauri dev
```

# 🛠️ 技术栈

- 编程语言：typescript + rust
- 前端框架：React
- 构建工具：vite
- css框架：tailwindcss
- 组件库：antd
- 代码格式化工具：prettier
- 桌面应用框架：tauri
